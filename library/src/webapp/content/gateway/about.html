درباره ساکای

ساکای مجموعه ای از ابزار های طراحی شده مختص استاد ها، محققان و دانش آموزان برای ایجاد محیطی جهت همکاری می باشد. با استفاده از یک مرورگر کاربران میتوانند از مجمرعه ای از ابزار ها جهت ایجاد سایت کاری موردنظر خود استفاده نمایند.

یک محقق میتواند سایت کاری را برای همکاری مشترک با همکاران خود بر روی یک پروژه ایجاد نماید.

یک استاد میتواند سایت کاری را ایجاد کند که به عنوان یک محیط قابلیت گفتگوی آنلاین در مورد یک تکلیف برای دانش آموزان را فراهم آورد.

یک دانش آموز میتواند محیطی را برای مطالعه جمعی یا فعالیت های فوق برنامه ای(طبق قوانین موسسه) با دیگر دانش آمرزان ایجاد نماید.
